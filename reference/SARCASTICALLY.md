# Speak sarcastically

Alternating caps for sarcastic tone.

## Usage

``` r
SARCASTICALLY(x)
```

## Arguments

- x:

  A character vector.

## Value

A character vector in alternating case.

## Examples

``` r
SARCASTICALLY("oh wow thats great")
#> [1] "Oh wOw tHaTs gReAt"
SARCASTICALLY("sure jan")
#> [1] "SuRe jAn"
```
