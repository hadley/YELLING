# Speak dramatically

Add dramatic pauses between words.

## Usage

``` r
DRAMATICALLY(x)
```

## Arguments

- x:

  A character vector.

## Value

A character vector with dramatic pauses.

## Examples

``` r
DRAMATICALLY("I cant believe it")
#> [1] "I... cant... believe... it"
DRAMATICALLY("no this cant be")
#> [1] "no... this... cant... be"
```
