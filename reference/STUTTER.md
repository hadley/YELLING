# Stutter text

Add stuttering effect to words.

## Usage

``` r
STUTTER(x)
```

## Arguments

- x:

  A character vector.

## Value

A character vector with stuttering.

## Examples

``` r
STUTTER("what do you mean")
#> [1] "w-what d-do y-you m-mean"
STUTTER("I dont know")
#> [1] "i-I d-dont k-know"
```
