# Yell louder

Add more exclamation marks to make text louder.

## Usage

``` r
YELL_LOUDER(x, n = 3)
```

## Arguments

- x:

  A character vector.

- n:

  Number of exclamation marks to add (default 3).

## Value

A character vector with added exclamation marks.

## Examples

``` r
YELL_LOUDER("HELLO!")
#> [1] "HELLO!!!!"
```
